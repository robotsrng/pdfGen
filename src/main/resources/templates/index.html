So the PDF Generator v1.0 works as so...

If you have alot of people filling out the same form, or sending you the same information that always goes onto the same paperwork, then this website may help you!

Simply make sure that the pdf form to be filled is built using named fields and upload it. Then, whenever you get an email or entry, either upload it as a txt file or copy and paste it and PDFGen will automatically fill your form and allow you to download it!

How neat right?

Caveats (of course there's cav eats): -The first line of your file or text has to be the PDF filename -The file or text has to be formatted like so:

[filename]
[fieldname] , [data]

Filename at the top followed by however many lines you want, as long as they are [fieldname][comma][data].

Once that is all set up (usually whatever email or form service can be set to these specs) you can start uploading and accessing your data all willy nilly!
